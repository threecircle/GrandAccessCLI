# GrandAccessCLI

```
Description:
  Command Line Interface for grant access manager

Usage:
  GrantAccessCLI [command] [options]

Options:
  --out-format <json|plain>  Output format
  --version                  Show version information
  -?, -h, --help             Show help and usage information

Commands:
  apply <file>          Apply manifest file
  set-context <server>  Set connection context
  status                Check status connect
```
